Risks to Manage
MBS XML parsing complexity
MBS schedule XML schema is not well-documented. Allocate 2 days to reverse-engineer the structure before writing the ETL. Use Services Australia's data dictionaries.
Azure PostgreSQL cold-start latency
Serverless tier can have cold-start delays. Use connection pooling (PgBouncer) from day one. Avoid spinning up new connections per API call.
Auth edge cases
Guest → account upgrade flow needs careful state management. Map localStorage data to DB on first sign-in. Test thoroughly before Iteration 2.
ABS concordance file size
Concordance CSV is large (~30MB). Pre-process and index by postcode at ETL time — do not load raw file into API response. Response time target: under 300ms.

Risks to Manage
RightDoor clinical boundary
Never suppress or downrank the ED option based on wait time alone. The comparison must always show ED as an available pathway. Risk of user harm if ED is hidden. Enforced at API layer, not just UI.
Healthdirect API rate limits
NHSD API has undocumented rate limits. Cache all provider searches by postcode + service type in PostgreSQL with a 24-hour TTL. Fall back to cached data gracefully if API is unavailable.
Cost estimate accuracy liability
Every cost display — including $0 bulk-billing estimates — must include the disclaimer "Estimate based on area bulk-billing rates. Confirm before attending." This is non-negotiable and enforced in the API response, not just frontend.
Calendar export compatibility
iCal (.ics) files have minor quirks across calendar apps. Test on Apple Calendar, Google Calendar, and Outlook. Use a tested library (ical-generator) rather than building from scratch.

Risks to Manage
PathwayAI hallucination boundary
This is NOT generative AI — it is pure vector similarity search over MBS text. The output is always a real MBS item, not generated text. "GP makes final eligibility decision" must appear on every AI result card. Confidence score shown to set expectations. Never claim the result IS the user's entitlement.
pgvector query performance
Target: under 500ms for a 384-dim cosine similarity search across 5,800 vectors. Use IVFFlat index with lists=100. Test with concurrent queries before launch. Fallback: pre-compute top-20 results for common lifestyle terms.
MBS quarterly re-index timing
MBS updates quarterly. Re-indexing pipeline must be triggered within 72 hours of MBS schedule release. Use Vercel Cron to check MBS update dates monthly. Do not serve stale embeddings without a "Last indexed: [date]" label.
Web Push API browser support
iOS Safari added Web Push in iOS 16.4 — check user agent and fall back gracefully to email-only reminders for older iOS devices. Never force push permission prompt on first load. Ask after user has seen value.
